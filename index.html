Installation Instructions

Windows

pip install pyside6
git clone -b ArkNote-Refresh --single-branch https://github.com/arc360alt/arcnote.git
cd arcnote
python arcnote.py

Linux

git clone -b ArkNote-Refresh --single-branch https://github.com/arc360alt/arcnote.git
cd arcnote
python3 -m venv venv
source venv/bin/activate
pip install pyside6
python arcnote.py
